# Derrick Context inspect

Command: `derrick context inspect`

Output context info.

@include "commands/context-inspect_desc.mdx"

## Usage

Usage: `derrick context inspect [<name>]`

Output information about a derrick context or general context info.

#### Global Options

- `-plain` - Plain output: no colors, no animation. The default is false.
- `-app=<string>` (`-a`) - App to target. Certain commands require a single app target for Derrick configurations with multiple apps. If you have a single app, then this can be ignored.
- `-project=<string>` (`-p`) - Project to target.
- `-workspace=<string>` (`-w`) - Workspace to operate in.

#### Command Options

- `-json` - Output information in JSON format. The default is false.
