# Derrick Token exchange

Command: `derrick token exchange`

Exchange an invite token.

@include "commands/token-exchange_desc.mdx"

## Usage

Usage: `derrick token exchange [options]`

Exchange an invite token for a normal token for login.

The "derrick token" commands are deprecated. They have been replaced with
the "derrick user" set of commands. Everything that was possible with
"derrick token" is now possible with "derrick user". For example,
"derrick token new" is now "derrick user token".

#### Global Options

- `-plain` - Plain output: no colors, no animation. The default is false.
- `-app=<string>` (`-a`) - App to target. Certain commands require a single app target for Derrick configurations with multiple apps. If you have a single app, then this can be ignored.
- `-project=<string>` (`-p`) - Project to target.
- `-workspace=<string>` (`-w`) - Workspace to operate in.

#### Command Options

- `-token=<string>` - The invite token to exchange.
