## files (registry)

Copies files to a specific directory.

### Interface

- Input: **files.Files**
- Output: **files.Files**

### Examples

```hcl
build {
  use "files" {}
  registry {
	use "files" {
	  path = "/path/to/file"
	}
  }
}
```

### Required Parameters

These parameters are used in the [`use` stanza](/waypoint/docs/derrick-hcl/use) for this plugin.

#### path

The filesystem path to store the files into.

- Type: **string**

### Optional Parameters

This plugin has no optional parameters.

### Output Attributes

Output attributes can be used in your `derrick.hcl` as [variables](/waypoint/docs/derrick-hcl/variables) via [`artifact`](/waypoint/docs/derrick-hcl/variables/artifact) or [`deploy`](/waypoint/docs/derrick-hcl/variables/deploy).

#### path

- Type: **string**
